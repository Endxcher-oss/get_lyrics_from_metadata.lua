# Features

Automatic Lyric Extraction: Automatically scans the metadata of audio files to search for fields containing lyric information.

Intelligent Matching: Supports matching any metadata field that starts with "lyrics" (case-insensitive).

Real-Time Display: Automatically loads and displays detected lyrics within mpv.

File Caching: Temporarily stores extracted lyrics in the user's runtime directory for easy management.

User Isolation: Each user has an independent lyric storage space to avoid permission conflicts.
